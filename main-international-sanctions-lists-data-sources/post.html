Introduction

Regulators ask to Financial Institutions for performing a systematic screening against lists of sanctioned individuals and organizations published by them in order to mitigate anti money laundering risks. For example:

  • Onboarding of new potential clients.
  • Stock of active clients.
  • The parties (originator, beneficiary, etc) of payments orders.

That is why it is important for the Financial Institutions to know where to find and understand the information of the sanctions lists published by the regulators.

There are three main sources of sanctioned individuals and organizations: United Nations (UN), European Union (EU) and Office of Foreign Assets Control (OFAC), the last one is the main source of USA sanctioned individuals and organizations.

UN, EU and OFAC occasionally change the organization of their websites and lists so the information provided in this post may not be accurate after such changes, especially the links. However, the general ideas provided in this post will still be a good reference.

UN

The UN sanctions lists can be found in United Nations Security Council Consolidated List. The page provides two different links in order to access the list content

  • HTML
  • XML

If the Financial Institution needs to integrate the UN list of sanctioned individuals and organizations with its information systems then the XML format is recommended. There is no schema available in UN website for the xml file, but the page United Nations Security Council Consolidated List provides a good description of the information contained in the xml file. Moreover, a deep inspection of the file allows to infer the schema.

EU

The entry point to the EU sanctions lists records is the page consolidated list of persons, groups and entities subject to EU financial sanctions. The list can be downloaded in several formats and versions. The most important formats in order to integrate the data with the Financial Institution information systems are:

  • CSV: version 1.0 and version 1.1
  • XML (only available in version 1.1)

The page consolidated list of persons, groups and entities subject to EU financial sanctions does not provide any link to the documentation of the lists in csv and xml formats but if you access to Financial Sanctions Files (FSF) website it is possible to download the schema of the xml version 1.1. However in order to access that page you have to create a account. Moreover, it is also possible to download the version 1.0 of the list in xml format.

OFAC

The OFAC landing page provides access to USA sanctions lists. The OFAC lists are splitted into two main groups: Specially Designated Nationals List (SDN) and Non-SDN Consolidated Sanctions List (Non-SDN). Within each group it is possible to download the list records in several formats. Among them the most relevant are:

  • CSV which is splitted into four files:
    • Main names: SDN and Non-SDN
    • Addresses: SDN and Non-SDN
    • Alternate names (aka): SDN and Non-SDN
    • Comments: SDN and Non-SDN
  • XML with several versions of the information:
    • Enhanced: SDN and Non-SDN
    • Advanced: SDN and Non-SDN
    • Finally there is other version with no specific name but I call it standard: SDN and Non-SDN

The documentation about the above formats can be accessed using the links:
